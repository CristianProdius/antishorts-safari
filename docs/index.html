AntiShorts

A Safari extension that removes YouTube Shorts from your browsing experience. No data collected. Runs entirely on your Mac.

Privacy Policy Support GitHub

Features

Homepage Clean

Removes Shorts shelves and Shorts items from your YouTube homepage feed.

Search Filter

Hides Shorts results from YouTube search so you only see regular videos.

URL Redirect

Direct /shorts/ links automatically redirect to the standard video player.

One-Click Toggle

Pause or resume blocking anytime from the Safari toolbar popup.

Sidebar Cleanup

Removes the Shorts navigation link from YouTube's sidebar menu.

Privacy First

No data collected, no analytics, no network requests. Everything runs locally.

How to Enable

  • Install the app from the Mac App Store
  • Open Safari → Settings → Extensions
  • Check the box next to "AntiShorts"
  • Grant permission for youtube.com when prompted
  • Visit youtube.com and enjoy a Shorts-free experience
